# MVP Scope

Included:
- Google OAuth authentication
- Projects with Owners and Contributors
- Modules inside projects
- Project and module behaviors
- Manual test cases (steps stored as JSON)
- Test runs lifecycle: STARTED -> IN_PROGRESS -> COMPLETED
- Locked test runs after completion
- Test execution with UNTESTED, PASSED, FAILED, BLOCKED and notes
- Assigning test runs to contributors
- Bugs linked to test cases and test runs
- Tasks (generic work items) with assignment, due dates, and comments
- Product Specs (rich-text) editable by owners, read-only for contributors
- UI References (image-based, owner managed)
- AI assistance: test case generation, test case improvement, module analysis
- Dashboard for assigned tasks and assigned test runs

Excluded:
- Payments or subscriptions
- Marketplace for paid testers
- Scrum or Agile workflows (sprints, epics, story points)
- CI/CD integrations
- Automation execution
- Enterprise RBAC
